This is the demo E-commerce website
Features of the product



Features for admin
* Can add Product and its Categories
* Can Manage Users Carts
* Can Add Product 
* Automatically Mail is sent to everyuser when New Product is added


Features for Users
* Can buy Produts on the basis of Categories
* Add To Carts
* Remove from Carts
* Can get amout of product after 'shiping in' on the basis of item selected
* Other Login,Register,Logout



Requirements to run the Project on your device :
    Create .env file and include the following things..
        * EMAIL_HOST_USER = 'Your email address'
        * EMAIL_HOST_PASSWORD = 'Your password for email host'
